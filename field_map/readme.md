Принцип работы:
Система состоит из нескольких подвижных объектов на платформе Raspberry Pi и одного ПК. Камеры на объектах считывают расстояние между видимыми соседними светодиодными метками (узнавая удаленность от меток) с частотой 10 Гц. Нагрузка на компоненты системы оптимизирована: на объектах при помощи камеры на RPi вычисляются только координаты меток в поле зрения камеры, далее эти координаты передаются по Wi-Fi на ПК, где производится расчет местоположения каждого отдельного объекта в общем пространстве и выдача управляющего воздействия для корректировки положения, если необходимо. Обновляемые местоположения всех объектов отображаются на карте помещения.
